# **Modelos**
Esta pasta armazena os outputs de um processo de treinamento. Caso o seu projeto envolva treinar modelos, é nesta pasta onde eles deverão ser disponibilizados.

> ⚠️ **Modelos grandes**
> Alguns arquivos de modelos podem ser grandes demais para serem inseridos diretamente no GitHub. Nesses casos, é possível utilizar o Git Large File Storage ou disponibilizar o modelo em plataformas como o HuggingFace. 
> Em todo caso, se o modelo foi disponibilizado, deve haver um link de acesso neste repositório.
